# 📚 Student Management System

A simple **Student Management System** built using **Flask (Python)** for backend and **HTML + CSS** for frontend.  
This project allows users to add and view student details easily.

---

## 🚀 Features

- Add student details (Name, Age, Course)
- View all students
- Simple and clean UI
- Backend using Flask
- Data stored temporarily in memory (Python list)

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/StudentManagementSystem.git
cd StudentManagementSystem
```

### 2️⃣ Create Virtual Environment

```
python -m venv env
```

Activate environment:

**Windows:**
```
env\Scripts\activate
```

**Mac/Linux:**
```
source env/bin/activate
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```
python app.py
```

Then open your browser and go to:

```
http://127.0.0.1:5000/
```

---

## 📌 How It Works

- `/` → Home page (Add student)
- `/view` → View student records
- Data is stored temporarily in a Python list (not stored in database).
